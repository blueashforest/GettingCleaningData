# Description of Column Data #

## First two Fields ##

    "activity_label" (col1) - Label of the activity performed, when the corresponding measurements were taken
    "subject" (col2) - Participant ID

### Activity Labels ###

    WALKING = 1
    WALKING_UPSTAIRS = 2
    WALKING_DOWNSTAIRS = 3
    SITTING = 4
    STANDING = 5
    LAYING = 6
